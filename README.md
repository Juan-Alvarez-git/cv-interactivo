# QA CV Interactivo

Proyecto estático independiente creado fuera de `rpg-game`.

## Archivos

- `index.html`
- `styles.css`
- `script.js`

## Cómo abrirlo

Puedes abrir `index.html` directamente en el navegador o levantar un servidor simple desde esta carpeta, por ejemplo:

```bash
cd /Users/imaginamos/Desktop/antigrativy/qa-cv-interactivo
python3 -m http.server 4173
```

Luego abre:

`http://localhost:4173`
